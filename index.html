시험 문제 : 연락처 사이트 만들기

  1. 연락처 목록 페이지와 연락처 읽기 페이지로 사이트를 구성하시오
  2. 각 페이지의 html, css, js를 분리해서 작성하시오
  3. header, nav, main, section, aside, footer 시맨틱 태그를 이용해 마크업하시오
  4. main 영역은 3단 레이아웃으로 작성하시오
  5. header, footer 영역의 내용은 문제를 참고하시오
  6. 구글 나눔 고딕 웹 폰트를 header 영역에 적용하시오
  7. nav 영역에는 index.html로 이동하는 버튼을 배치하시오
  8. section 영역의 구성은 문제를 참고하시오

제출

  1. github에 repository를 생성하고 프로젝트를 push하시오
  2. github repository를 netlify와 연동해 배포하시오
